# OPEN VS — Developer Security Toolkit

A single-file Python security toolkit with 116+ tools for developers, pentesters and security researchers.

## Features

- **116 tools** in one `.py` file — no installation needed
- Works on **Windows, Linux and Android/Termux**
- Uses **Python standard library only** (no pip required)
- Clean terminal UI with color output

## Categories

| Category | Tools |
|----------|-------|
| Network & Scanning | Port scan, ARP scan, DNS lookup, WHOIS, Traceroute |
| Web Security | SQLi tester, XSS finder, Admin bypass, CORS tester, HTTP headers |
| Passwords | Hash cracker, Password generator, Default creds tester |
| APIs & Auth | JWT analyzer, API security tester, OAuth checker |
| Minecraft | MC server scanner, plugin security checker |
| Crypto | Hash tools, Base64, Caesar cipher |
| OSINT | IP info, Domain checker, Email validator |
| System | Port listener, Environment scanner, Process checker |
| And much more... | |

## Usage

```bash
python open_vs.py
```

Then type a tool number (1–116) or search by name.

## Requirements

- Python 3.8+
- No additional packages needed
